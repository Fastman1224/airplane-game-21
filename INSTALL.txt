üìñ Installation and Setup Guide for Airplane Game v21

============================================================
                  üì¶ Install Dependencies
============================================================

Step 1: Install Python
    Make sure Python 3.8+ is installed
    https://www.python.org/downloads/

Step 2: Install Requirements
    Method A - Complete installation (Recommended):
        pip install -r requirements.txt

    Method B - Install only required packages:
        pip install pygame mediapipe opencv-python

    Method C - Complete installation (including NumPy):
        pip install pygame mediapipe opencv-python numpy


============================================================
                  üöÄ Run the Game
============================================================

Method 1 - No extra setup:
    python airplane.py

Method 2 - For better speed:
    pip install numpy
    python airplane.py

Method 3 - For best speed (Optional):
    py build.py
    python airplane.py


============================================================
            üîß C++ Acceleration (Optional)
============================================================

For best performance:

Windows:
    1. Download Visual C++ Build Tools:
       https://visualstudio.microsoft.com/visual-cpp-build-tools/
    2. Install Visual Studio Build Tools
    3. Run:
       py build.py

Linux:
    sudo apt-get install build-essential python3-dev
    python build.py

macOS:
    xcode-select --install
    python build.py


============================================================
          ‚ö†Ô∏è Common Problems and Solutions
============================================================

Problem: "No module named 'pygame'"
Solution: pip install pygame

Problem: "No module named 'mediapipe'"
Solution: pip install mediapipe

Problem: "No module named 'cv2'"
Solution: pip install opencv-python

Problem: Camera not working
Solution: Check camera is connected
         Make sure the app has permission to access it

Problem: Game is slow
Solution: pip install numpy

Problem: MSVC error
Solution: Don't worry! Game works without it
         If you want better speed, install Visual C++ Build Tools
